Project Description:

Secure, E-Commerce Flask Application

For this project, you’ll be working in groups of up to three people to develop a fully-fledged Flask web application, using everything you’ve learned this semester.

This project is for creating a web application with a backend database, secured forms, and a user management system. The users of your website should have a reason to make an account. For example, they might need to sign in to favorite an item, create a post, or purchase something.

  • Login/Registration w/ 2FA (Flask-Login)
  • CSP (Flask-Talisman)
  • CSRF-Protected Forms (Flask-WTForms)
  • Hashed Passwords
  • Bootstrap
  • Font Awesome Icons
